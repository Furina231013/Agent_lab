# AGENTS.md

This repository has a mandatory development rule:

- Read [DEVELOPMENT.md](/Users/huyh/learning/agent_lab/agent-lab/DEVELOPMENT.md) before making code changes.
- Follow the TDD workflow defined there by default.
- For feature work or bug fixes, write tests first, confirm red, then implement until green.
- After targeted tests pass, run the full test suite before closing the task.
- Keep `README.md` and `CHANGELOG.md` in sync when behavior or workflow changes.

If a task must deviate from TDD, state the reason clearly in the work summary.
